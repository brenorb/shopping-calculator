# User Guide

## Using the Calculator

### Basic Calculation

1. **Item Name**: Enter the product name (e.g., "Rice", "Olive Oil")
2. **Price**: Enter the total price as displayed in store
3. **Quantity**: Enter the package quantity/weight/volume
4. **Unit**: Select the appropriate unit from the dropdown
5. Tap **Calculate** to save the calculation

### Understanding Results

The app shows the **price per standardized unit**:
- Weight items → price per gram
- Volume items → price per milliliter  
- Count items → price per item

This normalization lets you compare different unit types easily.

### Supported Units

| Category | Units |
|----------|-------|
| Weight | g (grams), kg (kilograms), oz (ounces), lb (pounds) |
| Volume | ml (milliliters), L (liters), fl oz (fluid ounces), gal (gallons) |
| Count | count (individual items) |

### History View

The history list shows:
- Item name
- Original price and quantity with unit
- Calculated per-unit price
- Timestamp of calculation

History is sorted newest first. Use it to compare products while shopping.

## Examples

### Example 1: Comparing Rice Packages

**Package A**:
- Price: $5.99
- Quantity: 1
- Unit: kg
- Result: $0.00599 per gram

**Package B**:
- Price: $4.49
- Quantity: 800
- Unit: g
- Result: $0.0056125 per gram

Package B is cheaper per gram ($0.0056 vs $0.0060).

### Example 2: Comparing Juice Bottles

**Bottle A**:
- Price: $3.99
- Quantity: 64
- Unit: fl oz
- Result: $0.00211 per ml

**Bottle B**:
- Price: $6.99
- Quantity: 2
- Unit: L
- Result: $0.0035 per ml

Bottle A is cheaper per ml ($0.0021 vs $0.0035).
